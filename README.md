# Expense Tracker

A full-stack expense tracker application built using:

- React + Vite
- Express.js
- Node.js
- JSON-based persistence

## Features

- Add expenses
- View expenses
- Update expenses
- Delete expenses
- REST API backend
- File-based data persistence

## Tech Stack

Frontend:
- React
- Vite

Backend:
- Node.js
- Express.js

Tools:
- Postman
- Git & GitHub
